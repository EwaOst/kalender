# Herbal Collection Calendar

A responsive web application that helps users explore medicinal herbs and learn when and which parts of each plant can be collected throughout the year.

The application combines an interactive collection calendar with detailed information pages describing the characteristics, harvesting periods, and uses of selected herbs.

## Features

- Interactive herbal collection calendar
- Detailed pages for individual herbs
- Information about harvest periods
- Descriptions of roots, leaves, flowers, and fruits
- Responsive layout for different screen sizes
- Smooth scrolling navigation
- Image gallery with hover effects
- Parallax header animation
- Simple page transition animations

## Project Structure

The project consists of:

- HTML pages
- CSS styling
- JavaScript functionality
- jQuery animations
- Image resources

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- jQuery
- Google Fonts (Montserrat)

## Project Goal

The goal of the project was to design and develop a responsive website that presents information about medicinal herbs in an accessible and visually appealing way. The application focuses on user experience through animations, responsive design, and intuitive navigation.
